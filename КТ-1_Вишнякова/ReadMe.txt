1. Убедитесь, что исполняемый файл ProcessGenerator.exe и папка templates находятся в одной директории 
2. Дважды нажмите на исполняемый файл ProcessGenerator.exe для запуска программы
3. Введите имя и группу участника бизнес-процесса (lane)
4. Для добавления новых lanes нажмите кнопку "+ Добавить"
5. Чтобы очистить поля, нажмите "Очистить"
6. Для сборки par-файла нажмите "Запустить"
Готовый файл result.par будет находиться в папке \КТ-1_Вишнякова
