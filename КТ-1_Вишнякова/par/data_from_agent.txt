Это файл с описанием данных, которые нужно получить с помощью 
ИИ-агента, и их формата

1. Словарь variables для ролей и переменных
Ключами будут либо имена lane, либо названия переменных
Для lane в качестве значения будет словарь c format: 'Executor' 
и со значением исполнителя, если он был указан, иначе - пустая строка
Для переменной в качестве значения будет словарь cо значением 
формата переменной (текст, дата и т. п.), для ключа editor будет в качестве 
значения пустая строка

variables = {
    lane1: {
        format: 'some format', 
        editor: 'some editor'
        },
    lane2: {
        format: 'some format', 
        editor: ''
        },
    variable: {
        format: 'some format', 
        editor: ''
        },
    ...
}
